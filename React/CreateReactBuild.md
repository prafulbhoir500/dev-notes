#Config
```javascript
"scripts": {
  "dev": "vite",
  "build": "vite build",
  "build:uat": "vite build --mode uat",
  "build:prod": "vite build --mode production",
  "preview": "vite preview"
}

```


# Create Build
```bash
npm run build
```

# Create UAT Build
```bash
npm run build:uat
```

# Create Production Build
```bash
npm run build:prod
```
